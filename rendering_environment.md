# Figure Rendering Environment

- Python: 3.13 (project-local virtual environment)
- matplotlib: 3.11.0
- numpy: 2.5.1
- pillow: 12.3.0

The four main figures were rendered with the Python/matplotlib backend only. SVG/PDF text settings use `svg.fonttype = none` and `pdf.fonttype = 42`; TIFF output is 600 dpi and PNG output is 300 dpi.
